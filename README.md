# Terraform work

## An EC2 will be created 

To run -

* terraform init
* terraform validate
* terraform plan
* terraform apply

To teardown -

* terraform destroy

SSH file name is Estio-Ubuntu.pem and can be kept in .ssh folder

Make sure to have an .AWS/credentials file in your home directory

[estio]

aws_access_key_id = ""

aws_secret_access_key = ""
